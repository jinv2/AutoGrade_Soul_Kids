## 📖 项目简介 (Overview)
Shensist 魔法全学科工坊是一款基于 **Shensist Matrix** 架构的智能化、多学科、批量化智慧教学评测终端。它能够深度扫描学生的思维足迹，通过“因材施教”逻辑引擎，为每一个灵魂火花提供独一无二的独家画像。

The Shensist Soul Kids Workshop is an intelligent, multi-disciplinary, batch-processing evaluation terminal based on the **Shensist Matrix** architecture. It captures and analyzes students' thinking footprints, providing unique soul-dialogues and dimension-based profiles.

## ✨ 核心特性 (Features)
- **因材施教引擎 (Tailored Education Engine)**: 基于语义细节抓取算法，实现 10 个学生 10 种不同反馈。
- **批量矩阵处理 (Batch Matrix)**: 支持多名学生数据（姓名-答案格式）的一键并行评测。
- **魔法童话视觉 (Fairytale Aesthetic)**: V3.0 Ultimate 版视觉重塑，具备霓虹光晕、深空星云与拟态化设计。
- **响应式设计 (Mobile Ready)**: 100% 兼容手机与平板端，适应课堂移动教学场景。

## 🚀 快速启动 (Quick Start)
1. 下载本项目所有文件。
2. 双击打开 `index.html`。
3. 在“教学灵魂参考”中输入参考答案或目标。
4. 在“批量思维注入”中按格式（姓名-答案）输入学生数据。
5. 点击 **✨ 启动灵魂扫描 ✨**。
